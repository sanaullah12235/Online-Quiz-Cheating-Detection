# 📚 DSA Quiz Management System (C++)

A console-based Quiz Management System developed in C++ using Data Structures and File Handling. This project demonstrates the practical implementation of core Data Structures concepts such as Linked List and Stack in a real-world quiz application.

---

## 🚀 Project Overview

The DSA Quiz Management System allows teachers to manage quiz questions and students to attempt quizzes. The system automatically calculates scores, tracks time spent on each question, and detects suspicious behavior based on answer timing.

This project was built to strengthen my understanding of Data Structures and Algorithms (DSA), Object-Oriented Programming (OOP), and File Handling in C++.

---

## ✨ Features

### 👨‍🎓 Student Module
- Student login using name and registration number
- Attempt quiz questions
- Automatic score calculation
- Total quiz time tracking
- Time tracking for each question
- Cheating detection based on fast answers
- Result display after quiz completion

### 👨‍🏫 Teacher Module
- Secure teacher login
- Add new quiz questions
- Edit existing questions
- Delete questions
- View all questions
- View student results

### 💾 File Management
- Questions saved in `question_bank.txt`
- Student results saved in `student_results.txt`
- Automatic loading and saving of data

---

## 🧠 Data Structures Used

This project uses the following Data Structures:

- Linked List  
  → Used to dynamically store and manage quiz questions

- Stack  
  → Used to track time spent on each question for cheating detection

- Structures  
  → Used to represent questions and timing data

- Dynamic Memory Allocation  
  → Used for efficient memory management

---

## ⏱ Cheating Detection System

The system includes a basic cheating detection mechanism:

- Tracks time spent on each question
- Flags suspicious users who answer too quickly
- Displays cheating status as:
  - CLEAN
  - SUSPECTED
